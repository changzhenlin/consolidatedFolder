# 文件重复检查工具

这是一个简单的文件重复检查工具，可以帮助您扫描指定文件夹及其所有子文件夹，找出重复的文件名（包括文件类型后缀）。

## 功能特点

- 选择任意文件夹进行扫描
- 递归扫描所有子文件夹
- 自动识别并统计重复的文件名
- 显示重复文件的出现次数和位置
- 友好的用户界面，支持中文显示

## 如何使用

### 方法一：直接运行Python文件

1. 确保您的系统已安装Python 3.x
2. 直接运行以下命令：
   ```
   python file_duplicate_checker.py
   ```

### 方法二：打包为可执行文件（可选）

如果您希望创建一个无需安装Python的可执行文件，可以使用pyinstaller：

1. 安装pyinstaller：
   ```
   pip install pyinstaller
   ```

2. 创建可执行文件：
   ```
   pyinstaller --onefile --windowed file_duplicate_checker.py
   ```

3. 生成的可执行文件将位于`dist`文件夹中

## 使用步骤

1. 点击"浏览..."按钮选择要扫描的文件夹
2. 点击"开始扫描"按钮开始扫描过程
3. 扫描结果将显示在下方的表格中，包括文件名、重复次数和位置信息
4. 底部状态栏会显示扫描统计信息

## 注意事项

- 扫描大型文件夹可能需要一些时间，请耐心等待
- 为了保持界面整洁，每个文件最多显示前3个位置信息
- 工具只比较文件名（包括扩展名），不比较文件内容

## 技术说明

- 开发语言：Python 3.x
- 使用库：tkinter（GUI）、os（文件操作）、collections（计数器）
